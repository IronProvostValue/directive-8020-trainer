# Directive 8020 Trainer

A Lua-based training tool for Directive 8020 compliance simulations.

## Setup

Install Lua 5.3+ and LuaRocks. Then:

```
luarocks install --deps-only directive-8020-trainer-1.0-1.rockspec
```

## Usage

```bash
lua src/main.lua
```

## Tests

```bash
lua test/test_suite.lua
```
